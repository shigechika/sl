## sl - Steam Locomotive

When you mistype `ls`, a steam locomotive runs across your terminal.

Four versions: **1985**, **2010**, **2023**, **2026**.
The latest **2026** version detects existing text on the screen and
sweeps only when the train reaches it.

See [sl(1)](src/README.md) for full documentation.

## INSTALL

    brew tap tecolicom/tap
    brew install sl-classic

Toyoda Masashi's version is in Homebrew core: `brew install sl`
